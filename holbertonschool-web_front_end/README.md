I completed a project focused on learning how to structure a web page using pure HTML, without any CSS or visual styling. The goal was not to create a visually appealing site, but to understand the foundational building blocks of the web by using HTML5 correctly and precisely. Throughout the project, I practiced following proper HTML guidelines, paying close attention to details such as correct tag usage, semantic meaning, lowercase syntax, and overall document validity.

I learned how to build the skeleton of an HTML5 page and use semantic elements like header, main, footer, article, nav, section, and aside to organize content in a meaningful way. I also developed an understanding of when to use structural elements such as div versus inline elements like span, as well as the importance of maintaining proper heading hierarchy for accessibility and clarity.

In addition, I practiced creating lists, tables, and embedding different types of media, including images, audio, video, and external content. I explored the differences between common image formats such as SVG, GIF, PNG, and JPG, and learned how to structure data effectively using tables. A key requirement of the project was ensuring that all code is W3C-compliant and validated using the W3C Validator, reinforcing best practices for writing clean, standards-based HTML.
